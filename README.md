# Jenna & Lars Wedding Website

A personalized wedding website for our celebration in the Belgian Ardennes, June 26-28, 2026.

## Features

- **Interactive RSVP System**: Guests can RSVP directly through the website, powered by Supabase for real-time data management
- **Schedule Timeline**: Detailed weekend itinerary from arrival to departure
- **Travel Information**: Comprehensive guide with maps, parking, and transport options
- **FAQ Section**: Answers to common questions about accommodation, dress code, and more
- **Responsive Design**: Beautiful experience on both mobile and desktop

## Tech Stack

- **React** + **TypeScript** + **Vite**
- **Tailwind CSS** for styling
- **Supabase** for RSVP database and backend
- **Lucide React** for icons
- Hosted on **GitHub Pages**
